## 说明

这里存放与 **OI** 有关的代码。

+ 部分代码思路借鉴了题解。
+ 部分目录会有 `solution.md` 给出做题思路。

做题时间请参考 `Author Date` 而不是 `Committer Date` ，或者根据题目对应提交中注明的提交记录编号查找。
