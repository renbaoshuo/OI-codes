## 说明

本仓库存放与 **OI** 有关的代码。

- 部分代码的思路借鉴了题解。
- 部分题目文件夹下会有 `solution.md` 给出做题思路，也会有部分题目会在 [博客](https://oi.baoshuo.ren/) 中给出题解。
- 部分题目的 `data` 文件夹下会存放该题的部分测试数据。

做题时间请参考 `Author Date` ，或根据题目对应提交中注明的提交记录编号查找。
